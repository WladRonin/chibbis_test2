# chibbis_test2
My test case #2 for chibbis vacancy
  
---
  
1. Загружает данные о погоде в привязке к городам из prod.dbo.cities в таблицу dm.weather_series
2. Обновляет таблицу dm.orders_n_weather, обогащая данные заказов информацией из dm.weather_series

Данные погоды имеют часовую гранулярность.  
По умолчанию, обновляет данные за последние три дня.

(!) Ранний запуск может привести к ошибке, если данные еще не появились в системе

---
  
- Проект создавался в python 3.11.1 -- предположительно, он не будет работать с python < 3.11
- Для корректной работы необходимо после клонирования, в корне проекта создать файл .env:

```
DRIVER={ODBC Driver 18 for SQL Server}
SERVER=...
DATABASE=...
UID=...
PWD=...
```
- Т.к. это MS SQL Server, хост и номер порта должны быть разделены запятой, например SERVER=127.0.0.1,1433
  
- Для информации по флагам нужно выполнить "python main.py --help" в командной строке
  
---
  
Значимые детали:
- При загрузке данных по координатам, считается, что prod.dbo.cities.Lat -- это широта, a prod.dbo.cities.Lng -- это долгота  
- Данные о погоде забираются с учетом местного времени и сопоставляются с заказами по времени AddedOn -- считается, что это местное время
